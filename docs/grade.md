# 🎨 AI 智能调色风格 (Grade Styles)

SideLight 内置了多种由 Gemini AI 驱动的专业调色风格。
使用 `-s` 或 `--style` 参数来指定。

**使用示例:**

```bash
sidelight grade -s <风格名称> photo.jpg
```

---

## 🔹 基础 / 标准 (Base / Standard)

*通用摄影风格，专注于色彩校正和画质增强。*

| 风格名称           | 描述                                      | 适用场景             |
|:---------------|:----------------------------------------|:-----------------|
| **`natural`**  | **(默认)** 还原真实色彩，平衡曝光，校正白平衡。最自然的观感。      | 日常记录、旅行、新闻纪实。    |
| **`standard`** | 模拟标准相机配置文件。对比度适中，色彩饱和，细节锐利。             | 电商产品图、档案记录。      |
| **`vivid`**    | 鲜艳风格 (类似 Velvia 反转片)。增强色彩和对比度，画面通透且不失真。 | 自然风光、花卉、多彩建筑。    |
| **`flat`**     | 低对比度 (Log 风格)。最大化动态范围，保留所有高光和阴影细节，中性灰。  | 适合后续进行精细后期的专业用户。 |
| **`hdr`**      | 高动态范围。大幅提亮阴影，找回高光细节，增强局部对比度 (Clarity)。  | 高反差场景、室内摄影。      |

## 🔹 电影感 / 艺术 (Cinematic / Art)

*灵感源自电影色调和艺术潮流的风格化处理。*

| 风格名称               | 描述                                       | 适用场景          |
|:-------------------|:-----------------------------------------|:--------------|
| **`cinematic`**    | 电影质感。情绪化的光影，宽动态范围，极具故事感的调色。              | 叙事性摄影、情绪片。    |
| **`teal-orange`**  | 好莱坞大片风格。阴影偏青蓝 (Teal)，高光偏橙 (Orange)，肤色通透。 | 城市夜景、人像。      |
| **`cyberpunk`**    | 赛博朋克/霓虹风格。冷调/品红白平衡，高对比，强调青、粉、紫色。         | 霓虹灯夜景、未来感建筑。  |
| **`matte`**        | 哑光质感。提升黑点 (Black Point) 产生灰雾感，色彩柔和低饱和。   | 生活方式、情绪人像、静物。 |
| **`dreamy`**       | 梦幻柔焦。降低清晰度产生光晕 (Glow)，色彩粉嫩高调 (High-key)。 | 婚礼、梦幻写真、儿童。   |
| **`wes-anderson`** | 韦斯·安德森风格。粉彩配色，对称构图感，高饱和度但低对比度，温暖怪诞。      | 极简建筑、风格化写真。   |

## 🔹 胶片模拟 (Film / Analog Simulation)

*模拟经典胶片质感。*

| 风格名称            | 描述                                      | 适用场景           |
|:----------------|:----------------------------------------|:---------------|
| **`film`**      | 通用胶片感。加入颗粒，柔化高光，色彩浓郁，阴影微提。              | 怀旧风格摄影。        |
| **`kodak`**     | 柯达 Gold/Portra 风格。暖调，高光偏黄/红，肤色表现极佳，复古感。 | 阳光明媚的日子、人像、度假。 |
| **`fuji`**      | 富士胶片风格。强调绿色和冷色调，对比度较硬，阴影偏品红。            | 街头摄影、自然风光。     |
| **`polaroid`**  | 拍立得风格。褪色感，色彩偏移，柔焦，浓郁的复古氛围。              | 记忆片段、艺术快照。     |
| **`retro-70s`** | 70年代复古美学。强烈的黄/橙色偏，褪色的阴影，温暖怀旧。           | 复古时尚、黄金时刻。     |

## 🔹 黑白 (Black & White)

*单色转换风格。*

| 风格名称              | 描述                            | 适用场景        |
|:------------------|:------------------------------|:------------|
| **`bw`**          | 标准黑白转换。影调平衡自然。                | 通用黑白摄影。     |
| **`bw-contrast`** | 高反差 "Noir" 风格。深邃的黑，明亮的白，戏剧性强。 | 建筑光影、街头抓拍。  |
| **`bw-soft`**     | 柔调黑白。低对比度，灰阶丰富，梦幻感。           | 艺术微喷、柔美人像。  |
| **`bw-sepia`**    | 怀旧棕褐色 (Sepia)。                | 西部风情、历史感题材。 |

## 🔹 风景 / 环境 (Scenery / Environment)

*针对特定环境优化的调色方案。*

| 风格名称              | 描述                                 | 适用场景        |
|:------------------|:-----------------------------------|:------------|
| **`landscape`**   | 风光增强。强化绿色 (植被) 和蓝色 (天空)，细节丰富，对比有力。 | 自然风光、山川森林。  |
| **`golden-hour`** | 黄金时刻。强调日出日落的暖光，增强橙、红、黄色调。          | 日出日落时分。     |
| **`blue-hour`**   | 蓝调时刻。强调暮色中的深蓝冷调，保留阴影丰富度。           | 黎明、黄昏、城市夜景。 |
| **`urban`**       | 城市硬调。低饱和度 (保留红/黄)，高清晰度，强调混凝土质感。    | 街头、建筑、工业风。  |
| **`snow`**        | 雪景高调。校正白雪颜色 (去除灰/蓝)，高曝光，细节清晰。      | 冬季雪景、滑雪。    |

## 🔹 题材专属 (Subject Specific)

*针对特定拍摄对象优化。*

| 风格名称                   | 描述                             | 适用场景        |
|:-----------------------|:-------------------------------|:------------|
| **`portrait`**         | 通用人像。肤色红润通透，轻微磨皮，面部曝光准确。       | 肖像、家庭合影。    |
| **`portrait-glamour`** | 糖水/美颜人像。柔肤效果明显，高曝光，高光发光，唯美感。   | 时尚写真、美妆。    |
| **`food`**             | 美食摄影。暖色温，增加饱和度和锐度，让食物看起来更有食欲。  | 美食探店、菜谱。    |
| **`street`**           | 人文纪实。高对比度，颗粒感，强调叙事性和"决定性瞬间"。   | 街头摄影。       |
| **`macro`**            | 微距细节。极高锐度和纹理质感，色彩生动。           | 昆虫、纹理、静物特写。 |
| **`product`**          | 商业产品。中性白平衡 (纯白背景)，锐利清晰，色彩还原准确。 | 产品图、电商上架。   |
